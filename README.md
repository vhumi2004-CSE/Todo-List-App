# Todo-List-App
To-Do List Web App built using HTML, CSS, and JavaScript with task management and localStorage support.

The To-Do List Web Application is a simple and interactive task management tool developed using HTML, CSS, and JavaScript. The main objective of this project is to help users organize their daily tasks efficiently by providing a clean and user-friendly interface.

The application allows users to add new tasks, mark tasks as completed, and delete tasks when they are no longer needed. The structure of the application is created using HTML, while CSS is used to design the layout and improve the overall appearance. Styling elements such as colors, spacing, and custom checkbox icons are used to make the interface visually appealing and easy to use.

JavaScript plays a major role in implementing the functionality of the application. It is used for DOM manipulation, allowing dynamic creation of list items when a user adds a new task. Event handling is implemented to respond to user actions, such as clicking on a task to mark it as completed or clicking on the delete icon to remove it. The completed tasks are visually indicated using a line-through effect.

One of the key features of this project is the use of browser localStorage to store task data. This ensures that tasks are saved permanently in the browser and remain available even after refreshing or reopening the page. Functions are used to save and retrieve data efficiently.

During development, challenges such as handling dynamically created elements and managing user interactions were addressed using proper event delegation techniques.

Overall, this project demonstrates a strong understanding of basic web development concepts, including HTML structure, CSS styling, JavaScript logic, DOM manipulation, and data persistence, while delivering a practical and useful application.
